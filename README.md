# Worldy

Worldy is an addon onto the server WorldMC.org. This mod can be used to enhance your experience in the server as a player.

## Added aliases

| Alias     | Command of server |
|-----------|-------------------|
| `/global` | `/ch j global`    |
| `/local`  | `/ch j local`     |
| `/lc`     | `/ch j local`     |
| `/home`   | `/res spawn`      |
| `/bed`    | `/res spawn`      |
| `/vp`     | `/vparty`         |
| `/spawn`  | `/t spawn`        |
| `/st`     | `/t spawn`        |
| `/sn`     | `/n spawn`        |

## New commands

`/icemap`\
Opens [Nokta's ice highway map](https://nokteholda.github.io/WorldMC-Ice-Highways-Map).

`/maphere`\
Opens [the server's map](https://map.worldmc.org) on your current position in the server.

`/rule [number]`\
Displays a list of the server rules, if the number is given, displays the description.

`/stafflist`\
Displays a list of all server staff, with their online status & clickable option to run `/res <staff name>`.

`/vote`\
Overwrites the server's `/vote` command. It displays a shorter list with your username automatically filled in when you click it. The mod menu contains an option to auto open the links for you in your browser.

`/waypoint <option> <parameters>`\
A full functional waypoint system to display a raycast to a specific coordinate. Includes save/load ecosystem and a way to find your last death positon.
